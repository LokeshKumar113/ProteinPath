# ProteinPath-
ProteinPath is a location-aware app that helps users hit their daily protein goals on a budget. It tracks meals, protein intake, and spending, then recommends nearby restaurants and meals ranked by protein-per-dollar. Features include protein progress tracking, personalized meal suggestions, meal logging, and customizable nutrition goals.

## Features
- 📊 Track daily protein intake and food spending
- 📍 Location-aware restaurant recommendations
- 💰 Meals ranked by protein-per-dollar value
- 🎯 Set and monitor personalized protein goals

## How It Works
1. Set your daily protein goal and budget
2. Log meals throughout the day
3. Get nearby restaurant suggestions optimized for protein value
4. Stay on track without overspending
